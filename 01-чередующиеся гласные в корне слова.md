| Корень            | Правило          | Пример             | Исключения             |
| ----------------- | ---------------- | ------------------ | ---------------------- |
| **-гар- -гор-**   | без ударения — **О** | загора´ть — зага´р | изгарь пригарь выгарки |
| **-клан- -клон-** |                  | склони´ться        |                        |
| **-твар- -твор-** |                  | творе´ние          | утварь                 |
| **-зар- -зор-**   | без ударения — **А** | зарево, зарница, озарение, заревать|        |
